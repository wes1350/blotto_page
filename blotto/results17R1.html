Structure/Rules:
  • 7 battlefields, exactly 100 soldiers
  • Battlefields 1-4 worth 1 point each. Soldier #s on fields 1-4 must be non-increasing (e.g. 10,5,5,2 but NOT 2,3,0,1)
  • Battlefields 5-6 worth 2 points each. Soldier #s on fields 5-6 must be non-increasing (e.g. 20,10 or 10,10 but NOT 10,20)
  • Battlefield 7 worth 3 points
  • Solider #s must be nonnegative integers from 0 to 100
Examples of VALID submissions:
  • 100,0,0,0,0,0,0 (7 nonnegative integers which sum to 100; #1 >= #2 >= #3 >= #4; #5 >= #6)
  • 3,2,1,0,80,4,10
Examples of INVALID submissions:
  • 99,0,0,0,0,0,0 or 101,0,0,0,0,0,0 (not exactly 100 soldiers)
  • 45,45,5,1,1,1,1,1 (8 numbers, but only 7 battlefields exist)
  • 50.5, 49, 0.5, 0, 0, 0, 0 (some entries are not nonnegative integers)
  • 0, 1, 2, 3, 80, 4, 10 (On battlefields 1-4, # of soldiers is not non-increasing)
Example match between 2 players A, B:

A submits 14,14,14,14,14,15,15
B submits 0, 0, 0, 0, 25, 25, 50

A wins battlefields 1-4, winning 4 points (14 vs 0 each time)
B wins battlefields 5-7, winning 7 points (25 vs 14, 25 vs 15, 50 vs 15).
