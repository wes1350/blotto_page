Structure/Rules:
  • You can submit up to 3 entries. Submit all in the same google form!
  • 100 soldiers, 7 fields
  • field N worth N+5 (field 1 worth 6, ... , field 7 worth 12)
  • ties split the points on a field
  • entries must be in non-increasing order, except for up to one increase (e.g. 25, 20, 10, 10, 30, 5, 0)
Examples of VALID submissions:
  • 50, 35, 10, 0, 0, 0, 5 (7 nonnegative integers, sum <= 100, no more than one increase)
  • 17, 3, 3, 3, 40, 17, 17
Examples of INVALID submissions:
  • 0, 1, 2, 3, 4, 5, 85 (from left to right, entries increase more than once)
  • 99,0,0,0,0,0,0 or 0,0,0,0,0,1,100 (more than 100 soldiers)
  • 45,45,1,1,1,1,1,1 (8 numbers, but only 7 battlefields exist)
  • 50.5, 49.5, 0, 0, 0, 0, 0 (some entries are not nonnegative integers)
Example match between 2 players A, B:

A submits 25, 25, 0, 50, 0, 0, 0
B submits 14, 14, 14, 14, 14, 14, 15

A wins battlefields 1-3, winning 6+7+8 = 21 points. B wins fields 4-7, winning 9+10+11+12 = 42 points.
