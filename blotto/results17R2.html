Structure/Rules:
  • 7 battlefields, exactly 100 soldiers
  • If you win on battlefields 1-6, you get SQRT(# soldiers you put on that field) points
  • Battlefield 7 is worth double fields 1-6 (you get 2*SQRT(# soldiers) for winning)
  • If you tie, you get 0.5 * SQRT(# soldiers) [doubled for field 7]
  • Your entries on battlefields 1-6 must be non-increasing
  • Solider #s must be nonnegative integers from 0 to 100
Examples of VALID submissions:
  • 100,0,0,0,0,0,0 (7 nonnegative integers which sum to 100; #1 >= #2 >= #3 >= #4 >= #5 >= #6)
  • 25, 15, 10, 10, 5, 0, 35
Examples of INVALID submissions:
  • 99,0,0,0,0,0,0 or 101,0,0,0,0,0,0 (not exactly 100 soldiers)
  • 45,45,5,1,1,1,1,1 (8 numbers, but only 7 battlefields exist)
  • 50.5, 49, 0.5, 0, 0, 0, 0 (some entries are not nonnegative integers)
  • 0, 1, 2, 3, 80, 4, 10 (On battlefields 1-6, # of soldiers is not non-increasing)
Example match between 2 players A, B:

A submits 15,14,14,14,14,14,15
B submits 25, 25, 0, 0, 0, 0, 50

A wins battlefields 3-6, winning SQRT(14) * 4 points.
B wins battlefields 1, 2, 7, winning SQRT(25)*2 + SQRT(50)*2 points.
